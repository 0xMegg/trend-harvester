## 작업 로그

### 1. Skill에 프로젝트별 placeholder 추가 (20:21)

### 2. /review 커맨드 commit 순서 수정 — divebase 역반영 (20:38)

- **문제**: 어제(4/3) Reviewer 워크플로우에서 commit 순서를 고쳤지만, `/review` 커맨드 자체의 워크플로우 번호가 동기화되지 않았음
- **해결**: `.claude/commands/review.md`의 워크플로우 순서를 role-reviewer.md와 일치시킴
- handoff 작성 → commit 순서로 통일하여 APPROVE 후 uncommitted 파일이 남는 문제 완전 해결
